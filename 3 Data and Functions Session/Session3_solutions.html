  • We want a function that calculates the sum of the square roots of two numbers,

  • rounds it to two digits

    vyz calcu <- function(x,y) { root_x <- sqrt(x) root_y <- sqrt(y) result <- root_x + root_y rounded <- round(result, digits =2) return (round) }

  • and then does a check: when the result is bigger then a third number return: bigger then z, otherwise return: smaller then z

vyz calcu <- function(x, y, z) { root_x <- sqrt(x) root_y <- sqrt(y) result <- root_x + root_y rounded <- round(result, digits = 2) if (rounded >= z) { return("result is bigger then z") } else { return("result is smaller then z") } }
