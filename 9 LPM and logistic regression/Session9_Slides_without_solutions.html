LPM and logistic regression - Session 9

Recap

Binary variables

Binary variables are used when there are only two values that a variable can take.

Typical cases are:

  • all questions that can be answered with yes/no - “Do you have siblings?”, “Do you own a car?”

  • gender (when including only male/female and no other identities)

Representation in a functional context (in a linear regression):

  • Case 1 (Ownership = 1): \(y_i = \beta_0 + \delta \times 1 + \beta_1 \times x_i\)
  • Case 2 (Ownership = 0): \(y_i = \beta_0 + \delta \times 0 + \beta_1 \times x_i\)
  • Case 2 reduces to: \(y_i = \beta_0 + \beta_1 \times x_i\) (reference model)

Visual Representation:

Different types of models are used for different purposes:

Linear regression

A linear regression algorithm defines a linear relationship between independent and dependent variables.

It uses a linear equation to identify the line of best fit (straight line) for a problem, thereby enabling the visualization of the output of the dependent variables.

For example, linear regression may inform us about…

  • the influence of education on the income.

  • the influence of gender on grades.

  • the influence of a number of variables on one dependent variable.

Logistic regression

Logistic regression’s output lies between 0 and 1 as the algorithm is designed to predict a binary outcome for an event based on the previous observations of a data set.

It uses independent variables to predict the occurrence or failure of specific events.

For example, logistic regression predicts the chance that…

  • someone passes an exam based on their preparation time.

  • someone will get a credit based on their income.

  • an outcome will be achieved based on dependent variables.

Using the model for predictions:

coef_linear_model <- coef(linear_model_houses)
intercept_linear_model <- coef_linear_model[1]
b1_hat_linear_model <- coef_linear_model[2]  # coefficient for age 
b2_hat_linear_model <- coef_linear_model[3]  # coefficient for needs renovation

# Estimation of the price for a 10 year old house that needs renovation
age <- 10
renovation <- 1
renovation_needed <- ifelse(renovation == 1, "needs no renovation", "needs renovation")

predicted_price <- intercept_linear_model + b1_hat_linear_model * age + b2_hat_linear_model * renovation
predicted_price <- round(predicted_price, digits = 2)

print(paste("The house that is", age, "years old and", renovation_needed, ", would cost approximatly", predicted_price, "€."))

Solutions

Linear regression:

lm(formula = house_price ~ lot_size + garage_presence + house_age, data = data_houses)

Is a linear regression where the house price is explained by the size of the lot, the presence of a garage and the age of the house.

Logistic regression:

glm(formula = house_sold ~ lot_size + garage_presence + house_age,family = binomial, data = data_houses)

Is a logistic regression where the chances of the house being sold within the first 14 days after listing is explained by the size of the lot, the presence of a garage and the age of the house.
